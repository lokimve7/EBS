# Task 01-1 - 로컬 환영 서버 시작

## 설명
내 컴퓨터에서 서버를 실행했을 때 브라우저에서 `http://localhost:3001`로 접속할 수 있고, 접속한 페이지에 `환영합니다` 문구가 보이며, 서버 시작 시 터미널에서 서버 실행 안내 문구와 접속 주소를 함께 확인할 수 있어야 한다.

## 구현 항목
- [x] 서버를 실행하면 `localhost:3001`에서 접속할 수 있다.
- [x] 브라우저에서 `http://localhost:3001`에 접속했을 때 `환영합니다` 문구가 보인다.
- [x] 서버 시작 시 터미널에 서버 실행 안내 문구와 접속 주소가 함께 출력된다.
