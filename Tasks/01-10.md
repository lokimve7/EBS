# Task 01-10 - 이미지 탭 씬 카드 연동

## 설명
이미지 탭으로 이동했을 때 현재 화면의 스크립트 장면 데이터를 그대로 이어 받아 이미지 씬 카드 목록이 다시 구성되어야 한다. 스크립트 탭에서 만들어진 장면 수만큼 이미지 탭에도 카드가 보여야 하고, 각 카드에는 해당 장면 설명이 표시되어야 하며, 아직 생성된 이미지가 없는 상태가 기본으로 보이도록 맞춘다. 이때 이미지 탭 레이아웃에 배치된 기존 버튼 구성은 유지되어야 한다.

## 구현 항목
- [x] 현재 화면에 구성된 스크립트 장면 수를 기준으로 이미지 탭의 씬 카드 개수가 다시 구성된다.
- [x] 각 이미지 씬 카드에 대응하는 장면 설명이 표시된다.
- [x] 각 이미지 씬 카드의 이미지 영역은 처음 상태에서 이미지가 없는 빈 상태로 보인다.
- [x] 각 이미지 씬 카드의 기존 버튼 구성과 배치는 유지된다.

## 범위 밖
- 실제 이미지 생성 API 연결
- 이미지 파일 저장, 다시 불러오기, 다운로드 기능 구현
- 이미지 탭에서 장면 설명을 직접 수정하는 편집 기능
- 영상, 보이스, 조합 탭까지 같은 연동을 확장하는 작업

## 사전 전제
- `Tasks/01-6.md`의 스크립트 생성 결과로 장면 데이터와 씬 카드 렌더링 흐름이 준비되어 있어야 한다.
- `Tasks/01-7.md`의 이미지 탭 씬 카드 레이아웃과 버튼 구성이 준비되어 있어야 한다.
