# Task 01-14 - 비디오 탭 씬 카드 레이아웃

## 설명
비디오 탭으로 이동했을 때 `Layout/03_layout_video.png`와 유사한 외형의 장면별 영상 카드 목록이 보여야 한다. 전체 구성은 이미지 탭과 거의 같은 흐름을 유지하되, 카드의 미리보기 영역은 이미지 대신 영상이 들어갈 자리처럼 보여야 하고, 스크립트 장면 수에 맞춰 카드 개수와 장면 설명이 함께 이어져야 한다.

## 구현 항목
- [x] 비디오 탭의 하단 내용 영역에 장면별 영상 카드 목록 레이아웃이 구성된다.
- [x] 비디오 탭의 카드 목록은 현재 스크립트 장면 수를 기준으로 다시 구성된다.
- [x] 각 영상 카드에 대응하는 장면 설명이 표시된다.
- [x] 각 영상 카드의 미리보기 영역은 아직 생성된 영상이 없는 기본 상태로 보인다.
- [x] 각 영상 카드의 버튼 구성과 배치는 이미지 탭과 거의 같은 흐름으로 유지된다.

## 범위 밖

- 생성된 영상 파일 저장, 다시 불러오기, 다운로드 기능 구현
- 영상 미리보기 재생 컨트롤이나 타임라인 기능 추가
- 보이스, 조합 탭까지 같은 연동을 확장하는 작업
