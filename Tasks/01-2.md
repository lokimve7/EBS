# Task 01-2 - 공통 상단 탭 메인 화면 구성

## 설명
`Layout/layout_top.png`를 참고해 모든 탭에서 공통으로 사용하는 상단 영역과 탭 메뉴 외형이 있는 메인 화면을 만든다. 탭을 클릭했을 때 상단 영역과 탭 메뉴는 그대로 유지되고, 그 아래의 내용 영역만 현재 선택된 탭 기준으로 바뀌는 구조를 확인할 수 있어야 한다.

## 구현 항목
- [x] `Layout/layout_top.png`와 최대한 비슷한 상단 영역과 탭 메뉴 외형이 있는 메인 화면이 보인다.
- [x] 탭을 클릭했을 때 상단 영역과 탭 메뉴는 고정되고, 하단 내용 영역만 선택된 탭 기준으로 전환된다.
- [x] 현재 단계에서는 각 탭의 하단 내용 영역을 비워둔 상태로 유지할 수 있다.

## 범위 밖
- 각 탭별 실제 기능 구현
- 탭 하단 내용 영역의 상세 레이아웃 구성
- API 연동, 데이터 저장, 서버 기능 확장

## 사전 전제
- `Tasks/01-1.md`의 로컬 서버 실행 환경을 사용할 수 있어야 한다.
