# Task 01-3 - 스크립트 탭 하단 레이아웃 구성

## 설명
`Layout/01_layout_script.png`를 참고해 상단 공통 영역과 탭 메뉴 아래의 스크립트 탭 내용 영역을 `스크립트 설정`, `장면 구성` 두 영역으로 채운 화면을 만든다. 전체 페이지는 한 화면 안에 자연스럽게 들어와야 하며, `스크립트 설정` 영역의 내용은 영역 밖으로 넘치지 않고, 오른쪽 씬 카드 목록만 영역 안에서 세로 스크롤되는 상태를 확인할 수 있어야 한다.

## 구현 항목
- [x] `스크립트` 탭을 선택했을 때 상단 공통 영역과 탭 메뉴 아래에 `스크립트 설정`, `장면 구성` 두 영역이 `Layout/01_layout_script.png`와 유사한 비율로 배치되어 보인다.
- [x] 고정 크기가 꼭 필요한 요소를 제외한 주요 레이아웃이 픽셀 고정보다 비율 기반으로 구성되어, 전체 페이지가 세로로 길어지지 않고 한 화면 안에 자연스럽게 들어온다.
- [x] `스크립트 설정` 영역은 내용이 많아져도 영역 밖으로 넘치지 않는 외형을 유지한다.
- [x] 오른쪽 씬 카드 목록은 해당 영역 내부에서만 세로 스크롤되는 외형을 가진다.
- [x] 버튼, 입력창, 카드 등은 실제 동작 없이 화면 외형만 확인할 수 있으면 된다.
