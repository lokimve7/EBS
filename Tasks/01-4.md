# Task 01-4 - 스크립트 설정 입력과 생성 버튼 상태 전환

## 설명
`스크립트` 탭의 `스크립트 설정` 영역에서 사용자가 영상 주제를 직접 입력하고, 분위기와 스타일을 선택하며, 장면 수를 조절할 수 있게 한다. 또한 `스크립트 생성` 버튼을 눌렀을 때 버튼이 같은 모양으로 머무르지 않고, 스크립트 생성이 진행 중이라는 상태가 화면에 드러나야 한다.

## 구현 항목
- [x] `영상 주제`가 표시 전용 박스가 아니라 사용자가 직접 입력할 수 있는 입력 영역으로 보이고 동작한다.
- [x] `분위기`는 `감동적`, `신나는`, `유머러스` 중 하나를 선택할 수 있다.
- [x] `스타일`은 `다큐`, `브이로그`, `광고` 중 하나를 선택할 수 있다.
- [x] `장면 수`는 사용자가 현재 화면 안에서 선택 또는 조절할 수 있다.
- [x] `스크립트 생성` 버튼을 눌렀을 때 버튼 텍스트 또는 외형이 바뀌어 진행 중 상태가 드러난다.
