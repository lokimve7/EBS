# Task 01-5 - 공통 API 키 설정 창 구성

## 설명
`Layout/layout_api_key.png`를 참고해 상단의 `API 키 설정` 버튼을 눌렀을 때 현재 선택된 탭과 무관하게 공통으로 사용할 수 있는 API 키 설정 창이 열리게 한다. 이 창에서는 사용자가 API 키를 입력할 수 있어야 하고, `저장`을 누르면 입력한 키가 저장되며, `취소`를 누르면 창이 닫혀야 한다.

## 구현 항목
- [x] `API 키 설정` 버튼을 눌렀을 때 `Layout/layout_api_key.png`와 최대한 비슷한 API 키 설정 창이 열린다.
- [x] API 키 설정 창은 특정 탭 전용이 아니라 어느 탭에서든 같은 방식으로 열고 닫을 수 있다.
- [x] API 키를 입력할 수 있는 입력 영역이 보인다.
- [x] `저장` 버튼을 누르면 입력한 API 키가 저장된다.
- [x] `취소` 버튼을 누르면 입력 내용 변경 여부와 관계없이 API 키 설정 창이 닫힌다.

## 범위 밖
- 저장한 API 키를 실제 외부 서비스 호출에 사용하는 기능
- API 키 유효성 검사 또는 실제 발급 여부 확인
- API 키별 상세 안내 문구 확장이나 추가 서비스 연결
- 서버 측 암호화 저장소 구축

## 사전 전제
- `Tasks/01-2.md`의 공통 상단 영역에 `API 키 설정` 버튼이 준비되어 있어야 한다.
