# Task 01-6 - OpenAI API 기반 장면 스크립트 생성

## 설명
OpenAI API 공식문서를 기준으로 `스크립트 생성` 버튼을 눌렀을 때 현재 입력한 주제, 분위기, 스타일, 장면 수를 바탕으로 장면별 스크립트 데이터를 받아와야 한다. 응답에는 장면마다 한글 설명 요약, 한글 나레이션, 영상 길이, 장면 이미지 생성용 영어 프롬프트, 장면 영상 생성용 영어 프롬프트가 포함되어야 하며, 화면에는 받은 데이터 개수만큼 씬 카드가 생성되어 보여야 한다. 이때 영어 프롬프트 2종은 내부 데이터로만 유지하고 화면에는 표시하지 않는다.

## 구현 항목
- [ ] `스크립트 생성` 버튼을 누르면 현재 입력한 주제, 분위기, 스타일, 장면 수를 기준으로 OpenAI API에 스크립트 생성을 요청한다.
- [ ] OpenAI API 응답은 장면마다 `한글 설명`, `한글 나레이션`, `영상 길이`, `이미지 생성용 영어 프롬프트`, `영상 생성용 영어 프롬프트`를 포함해야 한다.
- [ ] 각 장면의 `영상 길이`는 3초에서 5초 사이로 생성되도록 요청하고, `한글 나레이션`도 해당 길이에 맞는 분량으로 생성되도록 요청한다.
- [ ] 응답으로 받은 장면 데이터 개수만큼 `장면 구성` 영역의 씬 카드가 새로 생성된다.
- [ ] 화면에는 각 장면의 `한글 설명`, `한글 나레이션`, `영상 길이`만 표시하고, `이미지 생성용 영어 프롬프트`와 `영상 생성용 영어 프롬프트`는 표시하지 않는다.

## 범위 밖
- 생성된 영어 프롬프트를 다음 단계 이미지/영상 API에 전달하는 기능
- 장면 카드별 재생성, 편집, 삭제 버튼의 실제 동작 연결
- 생성 결과 저장 이력 관리
- 프롬프트 품질 고도화나 다중 모델 비교

- `Tasks/01-5.md`의 API 키 설정 흐름 또는 현재 프로젝트에서 사용하는 OpenAI API 접근 설정이 준비되어 있어야 한다.
