# Task 01-7 - 이미지 탭 씬 카드 레이아웃

## 설명
`Layout/02_layout_image.png`를 참고해 이미지 탭의 하단 내용 영역을 장면별 이미지 카드 목록 화면으로 구성한다. 상단 공통 영역과 탭 메뉴 아래에서만 화면을 구성해야 하며, 전체 페이지 높이는 한 화면 안에 자연스럽게 들어오고 카드 목록만 내부에서 가로로 스크롤되어야 한다. 각 씬 카드는 장면 제목, 설명 문장, 이미지 미리보기 영역, 버튼 외형을 포함하되 실제 버튼 동작이나 이미지 생성 기능 연결은 이번 단계에 포함하지 않는다.

## 구현 항목
- [x] 이미지 탭을 선택했을 때 `Layout/02_layout_image.png`와 유사한 장면별 이미지 카드 목록 화면이 보인다.
- [x] 이미지 탭의 하단 내용 영역은 세로로 길어지지 않고 한 화면 높이 안에 자연스럽게 들어온다.
- [x] 씬 카드 목록만 내부에서 가로 스크롤되고, 페이지 전체는 불필요하게 세로 스크롤되지 않는다.
- [x] 각 씬 카드에는 장면 제목, 설명 문장, 이미지 미리보기 영역, `이미지 생성` 또는 `이미지 다시 생성` 버튼 외형, 보조 아이콘 버튼 외형이 보인다.
- [x] 고정 크기가 꼭 필요한 요소를 제외한 카드 배치와 영역 크기는 비율 기반으로 구성된다.

## 범위 밖
- 이미지 생성, 다시 생성, 다운로드 버튼의 실제 동작 연결
- 실제 이미지 파일 저장, 불러오기, 다운로드 처리
- 장면 카드 데이터와 이미지 생성 API 연동
- 이미지 탭 외의 다른 탭 레이아웃 변경

## 사전 전제
- `Tasks/01-2.md`의 탭 전환 구조가 준비되어 있어야 한다.
- `Tasks/01-6.md`의 장면별 스크립트 데이터 표시 흐름을 이후 이미지 카드 구성과 연결할 수 있는 상태여야 한다.
