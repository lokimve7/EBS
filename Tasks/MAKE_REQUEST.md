# 사용자 요청

- 아래 내용을 기반으로 `docs/PLANS.md`의 작업리스트 단계 하나를 작성해줘
- 그 이후에 해당 단계의 task 파일을 새로 만들어줘.
- 단, 아래 두 가지 경우에는 진행하지 말고 이유를 먼저 알려줘.
    - 이전 task 다음 순서로 오기에 적합하지 않을 때
    - 현재 task의 범위가 너무 클 때
 
## 원하는 동작
- 영상 탭에서 씬 카드마다 영상 생성 버튼을 누르면 해당 씬의 영상을 만들고 싶어.

## 세부 설명
- 영상 생성 버튼을 누르면 해당 씬의 이미지와 영상 생성 프롬프트를 EvoLink.AI 에 전달해서 영상 생성을 요청해야 해.
- EvoLink.AI 에서 Kling 을 사용할거야.
- 생성 중에는 해당 씬 카드가 로딩 상태로 보여야 해.
- 생성이 완료되면 씬 카드의 영상 영역에 결과 영상이 보여야 해.
