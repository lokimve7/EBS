# 사용자 요청

- 아래 내용을 기반으로 `docs/PLANS.md`의 작업리스트 단계 하나를 작성해줘
- 그 이후에 해당 단계의 task 파일을 새로 만들어줘.
- 단, 아래 두 가지 경우에는 진행하지 말고 이유를 먼저 알려줘.
    - 이전 task 다음 순서로 오기에 적합하지 않을 때
    - 현재 task의 범위가 너무 클 때
 
## 원하는 동작

- 스크립트 생성 버튼을 누르면 OpenAI API 로 장면별 스크립트를 받아와서 씬 카드로 보여주고 싶어.

## 세부 설명
- OpenAI API 공식문서를 확인하고 진행해.
- 입력한 주제, 분위기, 스타일, 장면 수 기준으로 OpenAI API 에 요청을 보내야 해.
- 응답 데이터는 장면마다 장면 요약 한글 설명, 한글 나레이션, 영상 길이, 장면 이미지 생성용 영어 프롬프트, 장면 영상 생성용 영어 프롬프트가 있어야 해.
- 받은 데이터 개수만큰 씬 카드가 화면에 생성되어야 해.
- 이미지 생성용 영어 프롬프트, 영상 생성용 영어 프롬프트 화면에 표시하지 않아야 해.
- 영상 길이가 3~5초 사이로 만들어지는 내용이 나오게 해 줘.
- 나레이션 길이도 설정된 영상 길이에 맞춰서 나오게 해 줘.
