# 사용자 요청

- 아래 내용을 기반으로 `docs/PLANS.md`의 작업리스트 단계 하나를 작성해줘
- 그 이후에 해당 단계의 task 파일을 새로 만들어줘.
- 단, 아래 두 가지 경우에는 진행하지 말고 이유를 먼저 알려줘.
    - 이전 task 다음 순서로 오기에 적합하지 않을 때
    - 현재 task의 범위가 너무 클 때
 
## 원하는 동작
- 스크립트 탭에서 설정값을 입력하고, 스크립트 생성 버튼을 누르면 버튼 상태가 바뀌게 하고 싶어.

## 세부 설명
- 주제를 입력할 수 있어야 해.
- 분위기는 감동적, 신나는, 유머러스 중에서 선택할 수 있어야 해.
- 스타일은 다큐, 브이로그, 광고 중에서 선택할 수 있어야 해.
- 장면 수를 선택할 수 있어야 해.
- 스크립트 생성 버튼을 누르면 버튼 모양이나 텍스트가 바뀌어야 해. (예 : "생성 중..." 같은 느낌)
